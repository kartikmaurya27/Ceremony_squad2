✨ Ceremony Squad – Premium Event & Wedding Planning Platform 🎉

Ceremony Squad Website

🌟 Project Overview

Ceremony Squad is a modern and visually stunning event management and wedding planning website created to deliver a luxury digital experience for users who want unforgettable celebrations and premium event services. The website combines elegant UI design, smooth animations, responsive layouts, and interactive elements to represent a professional event planning brand.

Built using HTML5, CSS3, and JavaScript, the project focuses on performance, creativity, and user experience. Inspired by modern luxury event companies, the website features cinematic visuals, dark-themed aesthetics, and engaging content sections that make the platform feel energetic, stylish, and premium.
